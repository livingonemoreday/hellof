selectorName description
link 元素链接地址未被访问时的样式

active 鼠标单击与释放间的样式
visited 元素链接地址已被访问过的样式

UI元素状态伪类选择器

  • E:hover | E:active | E:focus
  • E:enabled | E:disabled
  • E:read-only | read-write
  • E:checked:radio或checkbox元素处于选取状态下的样式
    E:default: 页面初始时默认处于选取状态的radio或checkbox元素的样式
    E:indeterminate: 页面初始时，若一组按钮无一配选中，则整组按钮的统一样式
  • E:selection: 指定当元素处于选中状态时的样式

结构伪类选择器

  • root: 页面根元素（即文档树中最顶层结构）的样式，HTML文档中则指<html></html>部分
:root {
    background: #2c3e50;
}
1
2
3
  • empty: 指定元素内容为空时使用的样式
:empty {
    background: #f08d49;
    height: 50px;
}
1
2
3
4
  • nth选择器
    • :first-child: 选择某个元素的第一个子元素
    • :last-child: 选择某个元素的最后一个元素
    • :first-of-type | last-of-type： 于:first-child|last-child同，特定元素类型
    • :nth-child(): 选择某个元素的一个或多个特定的子元素(n为固定字母，表示从0开始)
      • :nth-child(length)
      • :nth-child(n)
      • :nth-child(n*length)
      • :nth-child(n+length)：大于length前面的元素
      • :nth-child(-n+length)：小于length前面的元素
      • :nth-child(n*length+1)：隔length选一
    • :nth-last-child()：用法于:nth-child()同，从最后一个子元素开始计算来选择
    • :nth-of-type()：用法于:nth-child()同，选择特定类型的子元素
    • :nth-last-of-type()：用法于:nth-of-type()同，从最后一个特定类型的子元素开始计算选择
    • :only-child | only-of-type

属性选择器

  • E[attr]: 包含有属性attr的元素E
  • E[attr="v"]: 包含有属性键值attr=v的元素E
  • E[attr~="v"]: 属性attr的属性值为一个或多个词列表，只要有一个词和v相同
<div id="A">X</div>
<div id="B A">Y</div>
1
2
/*以上两个元素皆会被选中*/
div[id~="A"] {
   color: red;
}

  • E[attr^="v"]: 属性attr值为v开头的元素
  • E[attr$="v"]: 属性attr值为v结尾的元素
  • E[attr*="v"]: 属性attr值中包含v的元素
  • E[attr|="v"]: 属性attr值为v或v-开头的元素
