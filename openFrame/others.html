@RequestBody

需要把所有请求参数作为json解析，因此不能包含key=value这样的写法在请求url中，所有的请求参数都是一个json

  • 如果Controller中的形参未加@RequestBody，使用postman需key=value形式传参调用接口：
@PostMapping("/xx/create")
public CommonResult<Payment> create(Payment payment) {                         //①
    return restTemplate.postForObject(URL + "create", payment, Result.class);  //②
}
