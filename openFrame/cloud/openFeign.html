here we go

openFeign是在客户端以面向接口的方式调用目标服务，有集成Ribbon，所以是支持负载均衡的。

<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-openfeign</artifactId>
</dependency>

@SpringBootApplication
@EnableFeignClients
@RestController
public class ConsumerOpenFeign {

    public static void main(String[] args) {
        SpringApplication.run(ConsumerOpenFeign.class, args);
    }

    @Resource
    private FuckingService fuckingService;

    @GetMapping(value = "/get")
    public String get() {
        return fuckingService.get();
    }
}

超时控制

openFeign默认等待服务接口响应的时间为1s，超时即会抛错。修改等待时间

ribbon: #建立连接、建立连接后读取时间
  ConnectTimeout: 5000
  ReadTimeout: 5000
