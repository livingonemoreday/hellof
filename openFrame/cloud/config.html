The HTTP service has resources in the following form:

  • /{filename}/{profile}[/{branchname}]
    • http://localhost:8282/app/dev/master
  • /{filename}-{profile}.yml
    • http://localhost:8282/app-dev.yml
  • /{branchname}/{filename}-{profile}.yml
    • http://localhost:8282/master/app-dev.yml
  • /{filename}-{profile}.properties
    • http://localhost:8282/app-dev.properties
  • /{branchname}/{filename}-{profile}.properties
    • http://localhost:8282/master/app-dev.properties

目标仓库只有app-dev.yml，当访问.yml和.properties都是那个文件
